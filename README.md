# SIBUC - Sistema de Informações sobre Unidades de Conservação

## Sobre o Projeto

O **SIBUC (Sistema de Informações sobre Unidades de Conservação)** é uma aplicação web desenvolvida como trabalho da disciplina **Hands On Work VI** do curso de **Análise e Desenvolvimento de Sistemas da UNIVALI**.

O sistema tem como objetivo disponibilizar uma plataforma pública para consulta de unidades de conservação ambiental e registro de comunicações (denúncias, sugestões e informações) pela população, alinhado ao projeto de extensão **"Projeto Unidades de Conservação é Preciso"**.

### Objetivos

- Permitir a consulta pública de todas as unidades de conservação cadastradas
- Oferecer um canal para envio de comunicações sobre as unidades
- Exibir as comunicações ordenadas da mais recente para a mais antiga
- Promover a participação cidadã na gestão ambiental

## Funcionalidades

| Funcionalidade | Descrição |
|----------------|-----------|
| **Listagem de Unidades** | Exibe todas as unidades de conservação em cards com imagem, nome, localização, descrição e instituição responsável |
| **Nova Comunicação** | Formulário para envio de denúncias, sugestões ou informações sobre uma unidade específica |
| **Visualização de Comunicações** | Lista todas as comunicações enviadas, ordenadas da mais recente para a mais antiga |
| **Design Responsivo** | Interface adaptável para diferentes tamanhos de tela (desktop, tablet, mobile) |
